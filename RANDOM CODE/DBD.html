Catatan Desain Database:

Struktur database ini terdiri dari 3 tabel terpisah untuk memenuhi prinsip normalisasi:

  • Tabel VCD - Menyimpan informasi koleksi film
  • Tabel Anggota - Menyimpan data pelanggan
  • Tabel Transaksi Sewa - Mencatat aktivitas penyewaan

Relasi antar tabel dijaga melalui foreign key (Kode VCD dan Kode Anggota) untuk memastikan integritas data.
